CSS Flex

The flex box or flexible box model in CSS is a one dimensional layout model but has flexible and efficient layouts with distributed spaces.

Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

Properties for the Parent (Flex Container)

1) display: flex;

2) flex-direction: row;

3) flex-wrap: wrap;(flex-flow= flex-direction + flex-wrap;)

4) justify-content: center;

5) align-items: center;

6) align-content: space-between;

7) column-gap: 20px;

8) row-gap: 20px

9) gap: 20px;

Properties for the Children (Flex Item)

1) flex: 1; OR flex-grow: 1;

2) flex-shrink: 3;

3) order: 2;

4) align-self: center;
